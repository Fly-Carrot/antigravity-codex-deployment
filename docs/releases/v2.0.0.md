# v2.0.0 Release Notes

This release captures the current shared-fabric 2.0 baseline before any `multicli` integration work.

## Included in v2.0.0

- shared-fabric deployment and recovery scripts
- Gemini CLI shared knowledge base bootstrap source snapshot
- exact six-stage phase logger source snapshot
- compact dashboard v2 terminal monitor
- macOS floating dashboard source and app build script
- updated installer templates for Gemini settings paths
- test coverage for Gemini bootstrap and dashboard snapshot logic

## Deliberately excluded

- local secrets
- live receipts and handoff history
- exported state archives
- machine-specific caches and compiled artifacts

## Validation used for this release

- installer and Gemini bootstrap tests
- compact dashboard unit tests
- Python bytecode compilation checks
- local smoke validation for:
  - dashboard snapshot export
  - phase logger
  - macOS floating panel build and launch
  - Codex + Gemini dual-runtime dashboard activity

## What comes next

- `v2.1`: evaluate and integrate `multicli` as a runtime bridge MCP
- keep `global-agent-fabric` as the canonical shared knowledge base
- keep dashboard and phase logging attached to the shared fabric instead of the bridge layer
