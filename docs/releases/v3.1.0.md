# v3.1.0 Release Notes

v3.1.0 reframes the public product around the workflow this repository actually serves today: **Codex + Gemini on one canonical shared fabric**.

## Highlights

- refreshed public positioning around Codex + Gemini instead of older Antigravity framing
- a richer README with setup flow, screenshot-backed feature tour, and a recommended startup snippet
- new `Question Profile` surface in the desktop app for global user-profile distillation plus workspace overlay
- in-app setup assistant screenshot and documentation for new-machine bootstrap
- resilient desktop-app rebuild behavior even when SVG icon rasterization fails on the local machine

## Included in This Release

- canonical shared-fabric bootstrap and sync scripts
- user-question profile distillation, compilation, and workspace preload support
- desktop dashboard app source and packaged app build scripts
- dashboard snapshot export updates for the new profile surface
- screenshot-backed product documentation for GitHub

## Validation

- `python3 -m unittest discover -s tests -p 'test_*.py'`
- `python3 -m unittest discover -s tools/compact_dashboard/tests -p 'test_*.py'`
- `python3 -m py_compile fabric/scripts/sync/*.py install/*.py tools/compact_dashboard/*.py`
- repeated desktop app rebuild validation via `tools/compact_dashboard_desktop/build_dashboard_app.sh`

## Notes

- The app bundle is still named `Shared Fabric Dashboard`, while the public product narrative now emphasizes the Codex + Gemini pairing it supports.
- The canonical shared state remains `global-agent-fabric`; this repository continues to be the portable deployment snapshot, not the live memory store.
- Historical bridge metadata remains readable for compatibility, but it is not the main product story in this release.
