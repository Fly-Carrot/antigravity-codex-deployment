# v3.1.1 Release Notes

v3.1.1 is a polish release focused on making the product easier to understand on first contact and easier to use as a single-agent memory system.

## Highlights

- refreshed the GitHub homepage around a clearer product identity: **Shared Fabric Dashboard - Personal AI Memory Console for Codex and Gemini**
- moved the README story to a single-agent-first message, with Codex + Gemini collaboration framed as an upgrade rather than a requirement
- added logo-first README presentation, concise feature tour, badge-style metadata, and clearer setup flow
- documented the new readable Obsidian chat-history export path and manual export model
- prepared the desktop app bundle as `v3.1.1`

## Included in This Release

- shared-fabric bootstrap and workspace-enablement scripts
- desktop dashboard app source and build pipeline
- project memory, question profile, phase visibility, and sync audit surfaces
- manual Obsidian export for readable Codex and Gemini chat history
- updated GitHub documentation and screenshots

## Validation

- `python3 -m unittest tests.test_export_agent_chat_history tests.test_postflight_sync`
- `python3 -m py_compile tools/compact_dashboard/export_agent_chat_history.py`
- `tools/compact_dashboard_desktop/build_dashboard_app.sh`

## Notes

- The app bundle name remains `Shared Fabric Dashboard`.
- The canonical shared state remains your chosen `global-agent-fabric` root.
- The product works with a single runtime, while shared Codex + Gemini memory remains the higher-end collaboration path.
